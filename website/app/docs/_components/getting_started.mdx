# Getting Started

Welcome to **FIND-X**! 
It is a ready-made local search engine for your website. Follow these docs to integrate it effortlessly into your site.

## Key Features

1. Natural language-based search
2. Fast results
3. Low cost

## Prerequisites

Before getting started, ensure that you have:

- A website built with Next.js or React.js
- An API key from the [FIND-X Dashboard](https://find-x.tech/dashboard)
