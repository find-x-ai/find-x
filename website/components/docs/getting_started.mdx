# Getting Started

Welcome to the **Find-X** documentation! Follow the instructions below, and you'll be up and running in no time.

## What is Find-X?

Find-X is a cutting-edge conversational search engine designed specifically for websites. Unlike traditional search engines that rely on basic keyword matching, Find-X harnesses the power of your website's public data to deliver highly accurate search results.

The old way of searching websites is outdated and fails to meet the expectations of modern users. Find-X changes that by integrating the latest technologies into a seamless pipeline, making it easy to incorporate advanced search capabilities into your site.

Find-X is **not** a chatbot or AI assistant that simply interacts with users. Instead, it transcends those features by offering a more powerful, efficient search experience.

## What Makes Find-X Different?

While some AI assistants integrate with websites to answer user queries by gathering custom information, such as FAQs and documentation, **Find-X** takes a different approach. We focus on enhancing the search experience, overshadowing the need for an assistant.

With lightning-fast response times, Find-X directs users to the most relevant content on your site. The responses include source tabs, which indicate the context used by the AI to generate answers, along with relevant images from those sources.

But that’s not all. Find-X supports a wide range of content, including plain text, images, code snippets, and tables, ensuring users can access all the information they need in a format that works best for them.

Find-X is also **8X faster** than other AI service providers, delivering results in a fraction of the time. And with its **cost-effective pricing**, you'll enjoy premium performance without breaking the bank.
