# Desktop Application For Find-X
This application consists of all the management functionalities required to run find-x. This application is solely for the organizational purposes of the find-x project.

##### This application mainly consists of the following
- Crawler to handle the scraping.
- Request manager
- Log monitoring
- Usage monitoring

**Note** : Access to the application is strictly restricted to the core members of the project as it grants central access to the whole find-x project architecture.
